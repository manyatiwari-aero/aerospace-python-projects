# Aerospace Python Projects Python scripts for aerospace engineering calculations and simulation. Uses NumPy for array operations and Matplotlib for plotting. ## Scripts ### drag_table.py Calculates drag force at 5 different velocities using the drag equation. Prints a formatted table of results to the terminal. **Formula:** D = 0.5 × ρ × v² × Cd × A **Skills:** for loops, lists, f-string formatting ### drag_plot.py Plots drag force and lift force vs velocity (0–300 m/s) on a single graph. Saves the output as a PNG image. **Skills:** NumPy arrays, Matplotlib, savefig, linspace ## Requirements ``` pip install numpy matplotlib ``` ## About **Student:** [Manya Tiwari] | Aerospace Engineering, Year 2 **Skills practised:** Python, NumPy, Matplotlib, aerospace physics
